# C++ Projects

Примеры C++ проектов.

## Структура

```
C++/
├── project1/
│   ├── CMakeLists.txt
│   ├── src/
│   │   └── main.cpp
│   └── include/
│       └── header.h
└── project2/
    └── main.cpp
```

## Сборка

```bash
mkdir build
cd build
cmake ..
make
```

## Запуск

```bash
./project1
```
